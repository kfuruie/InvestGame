About Us
A project for UVA's CS 4640 Web PL, Spring 2020. InvestGame is a web app designed for novice investors to learn how to invest using fake currency.
Users will be able to buy, sell, and monitor simulated stocks without the risk of losing actual money.
Skills such as money management, risk assessment, and portfolio maintenance are important financial skills that we hope this app will help develop.
Happy Investing!
